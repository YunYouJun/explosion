# 关于

游戏规则有四条：

- 当周围仅有1个或没有存活细胞时， 原来的存活细胞进入死亡状态。（模拟生命数量稀少）
- 当周围有2个或3个存活细胞时， 网格保持原样。
- 当周围有4个及以上存活细胞时，原来的存活细胞亦进入死亡状态。（模拟生命数量过多）
- 当周围有3个存活细胞时，空白网格变成存活细胞。（模拟繁殖）
